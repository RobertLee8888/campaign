# Campaign

Parallel landing-page directions for the ALVA project, cloned from
Variant shared designs and tuned for production.

## Live pages

- **Direction A** — Intelligence Engine (dark cyber theme)
  → https://RobertLee8888.github.io/campaign/

- **Direction B** — Audit Your KOL Feed (light theme, dark cards, dark
  inverted closing) → https://RobertLee8888.github.io/campaign/audit.html

- **Campaign 3.0** — leaderboard-first iteration
  → https://RobertLee8888.github.io/campaign/campaign-3.0/

- **Campaign 4.0** — current sharing/marketing page (KOL cards,
  connect flow, results modal)
  → https://RobertLee8888.github.io/campaign/campaign-4.0/

Each link is an alternative exploration of the same product story,
not a parent / sub-page hierarchy.

## What's in here

- `index.html` — Direction A
- `audit.html` — Direction B
- `campaign-2.0/` — earlier exploration
- `campaign-3.0/` — leaderboard iteration
- `campaign-4.0/` — current sharing/marketing page
- `logo.svg` — shared brand mark, inlined where needed for re-tinting

## Run locally

No build step — open any HTML file in a browser, or serve the folder
with any static server (e.g. `python3 -m http.server`).
